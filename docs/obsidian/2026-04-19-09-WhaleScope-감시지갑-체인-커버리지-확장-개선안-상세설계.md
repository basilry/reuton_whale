### 0.4 비목표 (Non-goals)

- **BTC/XRP/DOGE에서 ETH와 동일한 이벤트 해상도 달성**: UTXO 체인은 구조적으로 "부분 관측"이 불가피하다. UX는 `is_partial_view` 플래그로 이 한계를 정직하게 드러낸다.
- **자체 풀노드 운영**: Bitcoin 풀노드 200GB+ 스토리지 유지는 본 프로젝트 범위 밖. 공개 API 게이트웨이로 해결한다.
- **실시간(subsecond) 스트리밍**: 현 파이프라인의 15분 cron 주기 내에서 동작해야 한다.

## §2 핵심 아키텍처 변경 — ChainCollector 인터페이스 일반화

### 2.1 현재 구조의 문제

`src/pipeline/common.py:300~331`은 **하드코딩된 2갈래 분기**다.

```python
# 현재 (문제)
if eth_collector is not None:
    for chain in EVM_CHAINS:
        addrs = [...]
        raw_events.extend(eth_collector.fetch(addrs, chain, ...))

if sol_collector is not None:
    sol_addrs = [...]
    if sol_addrs:
        raw_events.extend(sol_collector.fetch(sol_addrs, ...))

# 새 체인이 추가될 때마다 이 if 블록이 늘어남 → 확장성·테스트성 저하
```

문제점:
- 새 체인 추가 시 이 함수를 계속 편집해야 함(닫힘 원칙 위배).
- 각 collector의 `fetch` 시그니처가 미세하게 다름(`eth_collector`는 `chain` 인자 필요, `sol_collector`는 불필요). 추상화 부재.
- 지원 체인 enum이 흩어져 있음(`EVM_CHAINS`, `_CHAIN_IDS`, `_CHAIN_LABELS`, 시트 `chain` 컬럼, `signals.yaml` per-chain).

### 2.2 신규: `ChainCollector` 인터페이스 + Registry 패턴

**새 파일: `src/ingestion/base.py`**

```python
"""Common interface for all chain-specific collectors."""
from __future__ import annotations

from abc import ABC, abstractmethod
from typing import Iterable

from src.signals.models import Event


class ChainCollector(ABC):
    """Uniform interface for fetching on-chain events for a given chain."""

    #: Chain symbols this collector is responsible for.
    #: E.g. Etherscan → ("ETH","ARB","BASE","BSC","POLYGON"); XRPL → ("XRP",).
    supported_chains: tuple[str, ...] = ()

    #: True if this collector cannot guarantee observing all activity of a
    #: watched address (e.g., UTXO chains where one entity uses many addresses).
    is_partial_view: bool = False

    @abstractmethod
    def fetch(
        self,
        addresses: list[str],
        chain: str,
        since_ts: int,
        *,
        watched_index: dict | None = None,
        price_service=None,
    ) -> list[Event]:
        """Fetch events for the given addresses since `since_ts`.

        Collectors MAY raise collector-specific exceptions; the orchestrator
        catches and folds them into CollectedEvents.errors.
        """
```

**새 파일: `src/ingestion/registry.py`**

```python
"""Registry that maps chain symbols to the collector responsible for them."""
from __future__ import annotations

from collections import defaultdict
from typing import Iterable

from src.ingestion.base import ChainCollector


class ChainCollectorRegistry:
    def __init__(self) -> None:
        self._by_chain: dict[str, ChainCollector] = {}

    def register(self, collector: ChainCollector) -> None:
        for chain in collector.supported_chains:
            key = chain.upper()
            if key in self._by_chain:
                raise ValueError(f"Chain {key} already registered")
            self._by_chain[key] = collector

    def get(self, chain: str) -> ChainCollector | None:
        return self._by_chain.get(chain.upper())

    def supported(self) -> frozenset[str]:
        return frozenset(self._by_chain.keys())

    def group_addresses(
        self, watched_index: dict
    ) -> tuple[dict[str, list[str]], list[str]]:
        """Group watched addresses by collector; return (by_chain, unsupported_chains).

        Returns
        -------
        by_chain : {chain_symbol: [address, ...]}
        unsupported_chains : list of chain symbols that had addresses but no collector.
        """
        by_chain: dict[str, list[str]] = defaultdict(list)
        unsupported: set[str] = set()
        for addr, row in watched_index.items():
            chain = (row.get("chain") or "").upper()
            if not chain or chain == "EVM":
                chain = "ETH"  # legacy EVM fallback
            if chain in self._by_chain:
                by_chain[chain].append(addr)
            else:
                unsupported.add(chain)
        return dict(by_chain), sorted(unsupported)
```

### 2.3 `collect_recent_events` 리팩토링

```python
# src/pipeline/common.py (after)
from src.ingestion.registry import ChainCollectorRegistry

def _build_registry(env: PipelineEnv) -> ChainCollectorRegistry:
    registry = ChainCollectorRegistry()
    if env.etherscan_api_key:
        registry.register(EtherscanCollector(env.etherscan_api_key))
    if env.solscan_api_key is not None:
        registry.register(SolscanCollector(env.solscan_api_key or None))
    if env.flag_xrp and env.xrpl_api_base:
        registry.register(XrplCollector(env.xrpl_api_base))
    if env.flag_trx and env.trongrid_api_key:
        registry.register(TronCollector(env.trongrid_api_key))
    if env.flag_btc and env.btc_indexer_base:
        registry.register(BitcoinCollector(env.btc_indexer_base, api_key=env.btc_indexer_key))
    if env.flag_doge and env.doge_indexer_base:
        registry.register(DogecoinCollector(env.doge_indexer_base, api_key=env.doge_indexer_key))
    return registry


def collect_recent_events(
    *,
    sheets: SheetsClient,
    price_service: PriceService,
    registry: ChainCollectorRegistry,
    event_to_dict,
    since: datetime | None = None,
) -> CollectedEvents:
    since_dt = since or (datetime.now(timezone.utc) - timedelta(hours=24))
    since_ts = int(since_dt.timestamp())
    errors: list[str] = []
    raw_events: list[Event] = []

    try:
        watched_index = sheets.list_watched_addresses()
        by_chain, unsupported = registry.group_addresses(watched_index)

        if unsupported:
            msg = f"unsupported_chains={','.join(unsupported)}"
            logger.warning("Silent drop: %s", msg)
            errors.append(msg)

        for chain, addrs in by_chain.items():
            collector = registry.get(chain)
            if not collector or not addrs:
                continue
            try:
                raw_events.extend(
                    collector.fetch(
                        addrs, chain, since_ts,
                        watched_index=watched_index,
                        price_service=price_service,
                    )
                )
            except Exception as exc:  # per-chain isolation
                errors.append(f"{chain}:{exc}")
                logger.error("Collector error chain=%s: %s", chain, exc)

        # TG listener (unchanged)
        try:
            tg_rows = sheets.list_tg_whale_events(since=since_dt)
            if tg_rows:
                raw_events.extend(
                    tg_row_to_event(row) for row in tg_rows if isinstance(row, dict)
                )
        except Exception as exc:
            errors.append(f"list_tg_whale_events: {exc}")
            logger.error("Failed to load TG whale events: %s", exc)
    except Exception as exc:
        errors.append(f"collect_recent_events: {exc}")
        logger.error("Failed to collect recent events: %s", exc)

    chain_events = [event for event in raw_events if event.source == "chain"]
    transactions = [event_to_dict(event) for event in chain_events]
    return CollectedEvents(
        raw_events=raw_events,
        chain_events=chain_events,
        transactions=transactions,
        errors=errors,
    )
```

**이점**
- 새 체인 추가 = `ChainCollector` 구현체 + 레지스트리 등록 1줄.
- 체인별 오류는 `per-chain isolation`으로 전파 차단(한 체인 다운이 전체 파이프라인을 죽이지 않음).
- Silent drop 가드가 **구조적으로 내장**된다(`group_addresses`의 unsupported return).

```python
# src/pipeline/common.py (함수 추가)
def _summarize_chain_coverage(
    by_chain: dict[str, list[str]],
    unsupported: list[str],
    registry: ChainCollectorRegistry,

    return {
        "supported_chains": ",".join(sorted(registry.supported())),
```

```python
"""XRP Ledger collector via XRPSCAN public API."""
from __future__ import annotations

import requests

from src.ingestion.base import ChainCollector
from src.ingestion.normalizer import normalize_chain_tx, register_normalizer
from src.signals.models import Event
from src.utils.errors import XrplError
from src.utils.http_backoff import get_with_backoff
from src.utils.logger import get_logger

logger = get_logger("xrpl")

_BASE_URL = "https://api.xrpscan.com/api/v1"


class XrplCollector(ChainCollector):
    supported_chains = ("XRP",)
    is_partial_view = False  # account 모델

    def __init__(self, base_url: str = _BASE_URL) -> None:
        self._base_url = base_url.rstrip("/")

    def fetch(self, addresses, chain, since_ts, *, watched_index=None, price_service=None):
        events: list[Event] = []
        seen: set[str] = set()
        for addr in addresses:
            try:
                rows = self._fetch_account_tx(addr, since_ts)
            except XrplError as exc:
                logger.warning("XRPSCAN error addr=%s: %s", addr, exc)
                continue
            for row in rows:
                txid = row.get("hash") or ""
                if txid in seen:
                    continue
                seen.add(txid)
                row["_chain"] = "XRP"
                row["_watched_address"] = addr
                try:
                    events.append(normalize_chain_tx(row, "XRP", watched_index or {}, price_service))
                except Exception as exc:
                    logger.warning("normalize failed txid=%s: %s", txid, exc)
        return events

    def _fetch_account_tx(self, addr: str, since_ts: int) -> list[dict]:
        resp = get_with_backoff(
            do_get=lambda: requests.get(
                f"{self._base_url}/account/{addr}/transactions",
                params={"limit": 100}, timeout=15,
            ),
            is_rate_limited=lambda data: isinstance(data, dict) and data.get("error") == "rate_limit",
            error_cls=XrplError,
            logger=logger,
        )
        data = resp.json() or {}
        rows = data.get("transactions") or data if isinstance(data, list) else []
        # timestamp filter (XRPSCAN returns ISO; convert)
        from datetime import datetime, timezone
        filtered = []
        for row in rows:
            ts = row.get("date") or row.get("executed_time")
            try:
                dt = datetime.fromisoformat(ts.replace("Z", "+00:00")) if ts else None
                if dt and dt.timestamp() >= since_ts:
                    filtered.append(row)
            except Exception:
                filtered.append(row)  # defensive
        return filtered


@register_normalizer("XRP")
def _normalize_xrp_tx(raw, chain, watched_index, price_service) -> Event:
    from datetime import datetime, timezone
    ts_str = raw.get("date") or raw.get("executed_time")
    try:
        block_time = datetime.fromisoformat(ts_str.replace("Z", "+00:00")) if ts_str else datetime.now(timezone.utc)
    except Exception:
        block_time = datetime.now(timezone.utc)

    token = "XRP"
    amount_drops = float((raw.get("Amount") if isinstance(raw.get("Amount"), (int, float, str)) else (raw.get("Amount") or {}).get("value")) or 0)
    # XRPSCAN normalizes amounts in drops (1 XRP = 1_000_000 drops) for XRP; for IOUs it's human units.
    if isinstance(raw.get("Amount"), (int, float, str)):
        amount_token = amount_drops / 1_000_000.0
    else:
        amount_token = amount_drops  # IOU already human-readable

    from_addr = (raw.get("Account") or raw.get("from") or "").lower()
    to_addr = (raw.get("Destination") or raw.get("to") or "").lower()

    watched_address = (raw.get("_watched_address") or "").lower()
    direction = "out" if watched_address and watched_address == from_addr else "in"

    amount_usd = 0.0
    if price_service and amount_token and token == "XRP":
        price = price_service.get_usd("XRP")
        if price:
            amount_usd = amount_token * price

    counterparty_addr = to_addr if direction == "out" else from_addr
    cp_row = (watched_index or {}).get(counterparty_addr, {})

    return Event(
        source="chain",
        chain="XRP",
        tx_hash=raw.get("hash"),
        watched_address=watched_address or None,
        from_addr=from_addr,
        to_addr=to_addr,
        direction=direction,
        token=token,
        amount_token=amount_token,
        amount_usd=amount_usd,
        counterparty_category=cp_row.get("category"),
        block_time=block_time,
        collected_at=datetime.now(timezone.utc),
    )
```

## §7 Phase 3 — BTC 수집기 (핵심)

**공수**: 48시간. **선행**: P0, P2 (레지스트리 안정화). **브랜치**: `feat/chain-coverage-p3-bitcoin`.

### 7.1 UTXO 모델의 도전

Ethereum account 모델: "주소 A에서 B로 1 ETH" = 1 트랜잭션. 팔로잉 쉬움.

Bitcoin UTXO 모델: 트랜잭션은 **여러 inputs(UTXO 소비) + 여러 outputs(UTXO 생성)** 로 구성. "주소 A에서 B로 얼마" 같은 단순한 해석이 불가능하다. 한 트랜잭션이 관측 주소를 포함하면:
- 주소가 input 측에 있음 → **outflow** (이 주소에서 자금 빠짐)
- 주소가 output 측에 있음 → **inflow** (이 주소로 자금 들어옴)
- 양쪽에 다 있음(change 반환) → outflow만 세고 change는 제외

**결정**: Phase 3에서는 trade-off를 인정하고 **대표 주소 기반 관측 + `is_partial_view` 플래그**로 시작. 풀 클러스터링은 Phase 5+.

### 7.2 인덱서 선택

| 옵션 | 무료 한도 | 데이터 품질 | 채택? |
|---|---|---|---|
| **mempool.space API** (`https://mempool.space/api/address/{addr}/txs`) | 분당 ~30 req | 우수, 재계산된 inflow/outflow 제공 | **1차** |
| **Blockchair** (`/bitcoin/dashboards/address/{addr}`) | 1,440 req/day | 우수 | 2차(mempool 보완) |
| BlockCypher | 200 req/hour | 중간 | 보류 |
| QuickNode Bitcoin | 유료 | 최상 | 보류(비용 초과) |

mempool.space + Blockchair 조합으로 rate limit 회피. 시드 주소 10개 기준 15분마다 20~30 req이므로 mempool 단독으로도 여유.

### 7.3 `src/ingestion/bitcoin.py` 구현 스케치

```python
"""Bitcoin collector via mempool.space (primary) with Blockchair fallback."""
from __future__ import annotations

import requests
from datetime import datetime, timezone

from src.ingestion.base import ChainCollector
from src.ingestion.normalizer import register_normalizer, normalize_chain_tx
from src.signals.models import Event
from src.utils.errors import BitcoinError
from src.utils.http_backoff import get_with_backoff
from src.utils.logger import get_logger

logger = get_logger("bitcoin")

_MEMPOOL = "https://mempool.space/api"


class BitcoinCollector(ChainCollector):
    supported_chains = ("BTC",)
    is_partial_view = True  # !!! UTXO — 대표 주소만 감시 가능

    def __init__(self, base_url: str = _MEMPOOL, api_key: str | None = None) -> None:
        self._base = base_url.rstrip("/")
        self._headers = {"X-API-Key": api_key} if api_key else {}

    def fetch(self, addresses, chain, since_ts, *, watched_index=None, price_service=None):
        events: list[Event] = []
        for addr in addresses:
            try:
                txs = self._fetch_address_txs(addr)
            except BitcoinError as exc:
                logger.warning("mempool error addr=%s: %s", addr, exc)
                continue
            for tx in txs:
                if (tx.get("status") or {}).get("block_time", 0) < since_ts:
                    continue
                tx["_chain"] = "BTC"
                tx["_watched_address"] = addr
                try:
                    events.append(normalize_chain_tx(tx, "BTC", watched_index or {}, price_service))
                except Exception as exc:
                    logger.warning("BTC normalize failed txid=%s: %s", tx.get("txid"), exc)
        return events

    def _fetch_address_txs(self, addr: str) -> list[dict]:
        resp = get_with_backoff(
            do_get=lambda: requests.get(
                f"{self._base}/address/{addr}/txs", headers=self._headers, timeout=15,
            ),
            is_rate_limited=lambda d: False,  # mempool returns 429 HTTP
            error_cls=BitcoinError, logger=logger,
        )
        return resp.json() if resp.status_code == 200 else []


@register_normalizer("BTC")
def _normalize_btc_tx(raw, chain, watched_index, price_service) -> Event:
    """Convert mempool.space tx to Event with UTXO-aware inflow/outflow."""
    watched = (raw.get("_watched_address") or "").lower()
    status = raw.get("status") or {}
    ts = status.get("block_time") or 0
    block_time = datetime.fromtimestamp(ts, tz=timezone.utc) if ts else datetime.now(timezone.utc)

    inputs = raw.get("vin") or []
    outputs = raw.get("vout") or []

    spent_by_watched = sum(
        (inp.get("prevout") or {}).get("value", 0)
        for inp in inputs
        if (inp.get("prevout") or {}).get("scriptpubkey_address", "").lower() == watched
    )
    received_by_watched = sum(
        out.get("value", 0)
        for out in outputs
        if out.get("scriptpubkey_address", "").lower() == watched
    )

    # change detection: 같은 주소에 나간 것 중 일부가 돌아오면 change로 간주
    if spent_by_watched and received_by_watched:
        # 단순 휴리스틱: 돌아온 금액은 change. net 유출만 카운트.
        net = spent_by_watched - received_by_watched
        direction = "out" if net > 0 else "in"
        amount_sats = abs(net)
    elif spent_by_watched:
        direction, amount_sats = "out", spent_by_watched
    elif received_by_watched:
        direction, amount_sats = "in", received_by_watched
    else:
        direction, amount_sats = "in", 0  # degenerate

    amount_token = amount_sats / 1e8  # sats → BTC

    from_addrs = {
        (inp.get("prevout") or {}).get("scriptpubkey_address", "").lower()
        for inp in inputs
    } - {""}
    to_addrs = {out.get("scriptpubkey_address", "").lower() for out in outputs} - {""}

    # counterparty: 반대 방향의 첫 비-watched 주소
    counterparty = None
    if direction == "out":
        counterparty = next((a for a in to_addrs if a != watched), None)
    else:
        counterparty = next((a for a in from_addrs if a != watched), None)

    cp_row = (watched_index or {}).get(counterparty or "", {})

    amount_usd = 0.0
    if price_service and amount_token:
        price = price_service.get_usd("BTC")
        if price:
            amount_usd = amount_token * price

    return Event(
        source="chain",
        chain="BTC",
        tx_hash=raw.get("txid"),
        watched_address=watched or None,
        from_addr=next(iter(from_addrs), ""),  # 대표값
        to_addr=next(iter(to_addrs), ""),
        direction=direction,
        token="BTC",
        amount_token=amount_token,
        amount_usd=amount_usd,
        counterparty_category=cp_row.get("category"),
        block_time=block_time,
        collected_at=datetime.now(timezone.utc),
    )
```

```python
_ALIAS = {
    "bitcoin": "BTC", "btc": "BTC",
    "ethereum": "ETH", "eth": "ETH", "evm": "ETH",
    "solana": "SOL", "sol": "SOL",
    "ripple": "XRP", "xrp": "XRP",

    "arbitrum": "ARB", "base": "BASE", "bsc": "BSC", "bnb": "BSC",
    "polygon": "POLYGON", "matic": "POLYGON",

def canonical_chain(value: str) -> str:
    v = (value or "").strip().lower()
    return _ALIAS.get(v, v.upper())
```

모든 체인 문자열은 이 함수를 거쳐 canonical form으로 통일.

### 11.3 Failure mode 테스트

각 체인 collector에 대해:
- 4xx (rate limit, auth): 재시도 후 실패 → `errors`에 기록, 다른 체인 영향 없음.
- 5xx (서비스 장애): 재시도 3회 후 포기.
- 타임아웃: 15s 후 포기.
- 빈 응답: 에러 아님, 빈 리스트 반환.
- 기형 데이터(필수 필드 누락): `normalize_chain_tx`가 예외, `errors`에 warning.

## §17 오픈 이슈

1. **BTC 대표 주소의 "빙산 비율" 추정**: Binance BTC 전체 주소 수(~100k 추정) 중 감시 대표 8개가 커버하는 비율을 공개 라벨로 추정해야 UX 툴팁의 신뢰도가 올라간다. Arkham의 "Binance Bitcoin Total Holdings" 엔티티와 대표 주소 합의 차이로 역산 가능.
2. **Ripple Escrow 릴리스 예고 신호**: 매월 1일경 Ripple escrow에서 1B XRP가 unlock 스케줄. 이걸 "예고형" 카드로 만들 것인가, 아니면 실제 이동 시에만 스토리화할 것인가. 제품 결정.
3. **TG mirror의 neutral/조작 리스크**: 고신뢰 채널이라도 악의적 대형 이벤트 조작 가능성. 자체 관측과 5% 이상 금액 차이 시 경고 표시할지.
4. **시그널 per_chain_overrides 세분화 깊이**: 체인만 override할지, 체인+token(USDT on TRX) 조합까지 override할지. 후자가 정확하지만 config 복잡도 증가.
5. **Phase 3.5 UTXO 클러스터링 도입 시점**: 본 개선안 완료 후 언제 시작할 것인가. 외부 dataset(BitInfoCharts, Wallet Explorer) 스크레이핑 법적 리뷰 선결 필요.
6. **다국어 지원**: "부분 관측" 같은 전문 용어의 영어 카피(`Partial view — UTXO chain, representative address only`).
7. **TG listener가 BNB/TON 체인 이벤트를 이미 받고 있을 때** direct collector 부재 상태에서 `external_only_observation`만으로 충분한가, 아니면 BNB/TON도 Phase 5로 로드맵에 올려야 하는가.
8. **PoR 대조**: Binance·Coinbase의 Proof-of-Reserves 공개 주소와 우리 감시 주소의 교집합·여집합 분석 — 다음 개선안의 잠재 소재.

## §18 요약 — 실행 순서 한눈에

```
이번 주      PR-A (Registry 리팩토링)  →  PR-B (Silent drop 가드)

```

이 로드맵을 완수하면:
- 감시 체인 **2 → 6+**
- 감시지갑 **81 → 150+** (3개월), **300+** (6개월)
- ETH 편중 **96% → ≤ 70%** (3개월), **≤ 50%** (6개월)
- 침묵 실패 **해소**(service_health에 체인 커버리지 뱃지 상시 노출)
- 기획 #18(`curated_wallets`)의 chain enum이 **런타임에서 실제 의미를 가지게 됨**

본 개선안은 "제품이 고래를 지원한다"는 최소 기본기를 회복하는 동시에, 이후 시그널 품질·UX·스토리 다양성 확장의 **단일 진입점**이 된다.
