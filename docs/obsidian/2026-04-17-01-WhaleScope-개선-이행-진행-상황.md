## 이슈 및 해결

| 이슈 | 원인 | 해결 |
|------|------|------|
| Worktree race condition (2/3 실패) | `.git/config` lock 동시 접근 | 순차 재시도 |
| `.claude/worktrees/magical-diffie` embedded repo staged | 실수로 `git add` | `git rm --cached` + `.gitignore` 등록 |
| 에이전트들이 plan mode 분석만 하고 구현 안 함 | 권한 거부 (Write/Edit/Bash 차단) | 메인에서 직접 구현으로 전환 |
| `pytest --timeout` 플래그 미인식 | 프로젝트 pytest config 미정의 | 플래그 제거 |
| `_RATES` 형식 가정 오류 | 계획서는 dict, 실제는 tuple `(input, output)` | 튜플 형식 유지하며 추가 |
| `ANALYSIS_LOG_HEADERS` 구조 가정 오류 | 계획서와 실제 필드 순서/이름 차이 | 실제 구조에 맞춰 `status` 삽입 |
