# WhaleScope 목표 아키텍처 구현 프롬프트 (Claude Code용)

아키텍처 문서의 12개 Action Items를 Claude Code가 실행 가능한 6개 트랙으로 재편. 트랙은 **의존성 순서**로 배열되며, 각 트랙 종료 시 테스트·린트·git 상태 확인 후 다음 트랙으로 넘어간다.

---

## 사용법

1. Claude Code 세션 시작 후 프로젝트 루트(`02015_reuton_whale`)에서 실행.
2. 먼저 **프리플라이트 프롬프트**로 작업 브랜치·백업·의존성 설치 확인.
3. 트랙 0~5를 순서대로 붙여넣기. 각 트랙 완료 후 사용자가 코드 리뷰/테스트 통과 확인.
4. 트랙은 독립 PR로 분할 가능.
